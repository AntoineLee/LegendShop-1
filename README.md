## LegendShop
LegendShop是一款基于react-native的电商类app，现已支持安卓和ios端

## 完成功能
- [x] 首页
- [x] 分类
- [x] 搜索
- [x] 商品详情
- [x] 个人中心
- [ ] 购物车
- [ ] 订单管理
- [ ] 积分红包

## 工程运行
```
$ git clone https://github.com/xiayumoGit/LegendShop.git
$ cd LegendShop
$ npm install
$ react-native run-ios / react-native run-android
```
